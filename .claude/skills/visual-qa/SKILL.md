# Visual QA Skill

## Rol

Actúa como QA visual senior y design reviewer.

## Objetivo

Detectar problemas visuales antes de avanzar de fase.

## Checklist

Revisar:

- Layout general.
- Jerarquía visual.
- Espaciado.
- Alineación.
- Responsive.
- Contraste.
- Estados hover/focus.
- Imágenes rotas.
- Iconos rotos.
- Texto fuera de flujo.
- Animaciones excesivas.
- Dependencia crítica de JavaScript.
- Elementos que parecen plantilla genérica.
- Inconsistencias con el estilo dark/minimal/técnico.

## Reglas

- No rediseñar automáticamente.
- No modificar código salvo que `MIGRATION_TASK.md` lo permita.
- Reportar problemas concretos.
- Proponer correcciones pequeñas.
- Priorizar problemas visibles y medibles.
